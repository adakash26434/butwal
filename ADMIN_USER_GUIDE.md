## 📄 MANAGE PUBLIC PAGES (New Feature!)

### Go to: Admin → **Page Content (CMS)**

This is the most powerful new feature! You can now edit all public website content without touching any code.

#### What You Can Edit:
- **Homepage**: Hero title, subtitle, features section, call-to-action buttons
- **About Page**: Company mission, vision, team leadership profiles
- **Services Page**: Main services description and content
- **Careers Page**: Job culture, company values, why join us message
- **Contact Page**: All contact information and social links
- **Footer**: Footer text, copyright, links
- **Global Settings**: Site title, tagline, logo URL, theme colors

#### How to Use:
1. Click "Page Content (CMS)" in left admin menu
2. Select the page you want to edit from tabs
3. Fill in the fields with your content
4. Click "Save"
5. Changes appear on the website instantly ✅

**No coding needed. No deployment needed. No waiting.**

## 🛍️ MANAGE PRODUCTS

### Go to: Admin → Products

Create and manage all your software products.

#### Creating a New Product:
1. Click **+ New Product** button
2. Fill in these tabs:

   **Basic Tab:**
   - Product Name (required)
   - Tagline (short one-liner)
   - Icon / Emoji (optional, for visual appeal)
   - Lucide Icon (technical icon name)
   - Icon Color (choose blue, teal, purple, etc.)
   - Price From (optional - leave blank for "Custom pricing")
   - Category (Banking Software, Mobile App, etc.)
   - Badge (Popular, New, Flagship, etc.)

   **Content Tab:**
   - Summary (short description)
   - Description (detailed explanation)
   - Features (bullet points, one per line)
   - Highlights (special features)
   - Demo Screenshot URL (link to screenshot)

   **Homepage Tab:**
   - Show on Homepage? (checkbox)
   - Position on homepage (1, 2, 3...)
   - Card Style (wide, dark theme, custom CSS)

3. Click **Create Product**

#### Editing a Product:
1. Find the product in the list
2. Click the edit (pencil) icon
3. Make changes
4. Click **Update Product**

#### Deleting a Product:
1. Click the trash icon
2. Confirm deletion
3. Done ✅

## ⚙️ MANAGE SERVICES

### Go to: Admin → Services

Similar to products, but for your services/solutions.

#### Creating a Service:
1. Click **+ Add Service**
2. Fill in:
   - Service Title (required)
   - Tagline
   - Icon (with interactive picker)
   - Price From (optional)
   - Features (list of service features)
   - Highlights (key benefits)
   - Badge (e.g., "Popular")
   - Screenshot (visual representation)
   - Position (order in list)
   - Active (show/hide)
3. Click **Create Service**

#### Tips:
- Click the icon button to browse and select from hundreds of professional icons
- Features and highlights: One per line, each becomes a bullet point
- Price field: Leave blank for "Custom pricing" or enter a number

## 🖼️ MANAGE BANNERS & SLIDERS

### Go to: Admin → Banners

Create banners for homepage hero slider and site announcements.

#### Creating a Banner:
1. Click **+ New Banner**
2. Select "Use For":
   - **Homepage Hero Slider** = Big photo carousel at top of homepage
   - **Site-wide Announcement** = Shows on all pages
   - **Specific Page** = Shows only on that page

3. Fill in:
   - Title (headline text)
   - Subtitle (supporting text)
   - Style (Info, Success, Warning, Danger, Promo)
   - Link URL (where banner links to - optional)
   - Image URL (banner photo - optional)
   - Button Text (CTA button - optional)
   - Position (order/sequence)

4. Click **Create Banner**

#### Homepage Hero Slider:
- Multiple banners with "Homepage Hero Slider" automatically rotate every 5.5 seconds
- Add 3-5 banners for best effect
- Upload high-quality images (minimum 1200x400 pixels)

## 👥 MANAGE TEAM MEMBERS

### Go to: Admin → Team

Add and manage your team/staff directory.

#### Adding Team Member:
1. Click **+ Add Member**
2. Fill in:
   - Full Name (required)
   - Role/Title (CTO, Designer, etc.)
   - Bio (background and expertise)
   - Photo (upload or link)
   - Email (optional)
   - LinkedIn URL (optional)
   - Position (order in team listing)

- Photos work best at 400x400 pixels
- LinkedIn URLs should be like: `https://linkedin.com/in/username`
- Leadership team members appear in a special section

## ✅ BEST PRACTICES

### For All Forms:

1. **Read the Hints** 📝
   - Each field has a hint below it explaining what it does
   - Follow character limit guidelines
   - Use provided placeholders as examples

2. **Use Proper Formatting** 🎨
   - Keep titles concise (< 100 characters)
   - Use line breaks in features/highlights (one per line)
   - Use professional language
   - Avoid excessive punctuation

3. **Add Images** 📸
   - Products: 400x300 minimum
   - Banners: 1200x400 minimum  
   - Team: 400x400 square images
   - Gallery: 1200x800 minimum
   - Use HTTPS URLs for all images

4. **Keep It Organized** 📋
   - Use consistent naming conventions
   - Add position numbers to control order
   - Archive old content instead of deleting
   - Use slugs consistently

5. **Review Before Publishing** 🔍
   - Check on actual website after saving
   - Verify links work
   - Ensure images display correctly
   - Proof-read text for typos

## ⚡ TIPS & TRICKS

### Keyboard Shortcuts:
- **Tab** = Move to next field
- **Shift+Tab** = Move to previous field
- **Enter** = Submit form (when focused on submit button)
- **Escape** = Cancel/close dialogs (usually)

### Mobile Admin:
- ✅ All forms work on phone/tablet
- ✅ Responsive design adapts to screen size
- ✅ Touch-friendly buttons
- ✅ Same features as desktop

### Bulk Actions:
- Edit one item at a time
- Use position numbers to reorganize
- Archive instead of delete for safety

### Common Errors:
- **"required field missing"** = Fill all required fields (marked with *)
- **"invalid URL"** = Use full URLs with https://
- **"Save failed"** = Refresh page and try again
- **Price field showing "Custom"** = Leave price blank (it's intentional)

## 📞 TROUBLESHOOTING

### Form won't save?
- Scroll up to see error message
- Fill all red * required fields
- Check character limits (too long?)
- Try refreshing page and again

### Changes not showing on website?
- Hard refresh: Press Ctrl+Shift+R (Cmd+Shift+R on Mac)
- Clear browser cache
- Wait a few seconds (changes usually instant)
- Check that item is marked "Active"

### Images not uploading?
- Use HTTPS URLs, not HTTP
- Make sure image still exists at URL
- Try a different image
- Check file size (not too large)

### Can't find something in admin?
- Use browser search: Ctrl+F / Cmd+F
- Check other tabs at top of page
- Look in left navigation menu
- Try scrolling down
